# 프로젝트 설명
구현해보고 싶은 기능을 구현하며 안드로이드, 스프링을 학습하는 프로젝트입니다. 아키텍처, 디자인 패턴 등 자유롭게 적용해보고 미래의 저를 위해 투자하는 시간을 가져보겠습니다.


## 현재까지 구현된 기능
- 블로그 글 전체 조회
- 블로그 글 생성


## 다음으로 구현할 기능
### 안드로이드
- 블로그 글 삭제
- 블로그 글 수정

### 스프링
- 회원가입, 로그인
